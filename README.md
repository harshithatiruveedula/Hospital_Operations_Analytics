🏥 Hospital Operations Analytics Dashboard

An interactive Hospital Operations Analytics Dashboard built using **Microsoft Excel 2010** to analyze **55,500 patient records**. The dashboard provides insights into patient demographics, departmental performance, revenue, admissions, and hospital operations using Pivot Tables, Pivot Charts, KPI Cards, and Slicers.

📊 Project Features

- Interactive Dashboard
- KPI Cards
- Pivot Tables
- Pivot Charts
- Slicers for Dynamic Filtering
- Department-wise Revenue Analysis
- Patient & Admission Analysis
- Medical Condition Analysis
- Age Group Analysis

## 🛠 Tools & Skills Used
